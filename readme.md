# Schedule App
Schedule App is an App designed by Mavhungu Ronewa using Nodejs

## Getting Started
1. Clone the repository
2. run `yarn install`

### Prerequisites
```
1. NodeJS
2. MongoDB
```
### Installation
```
$ yarn install
$ yarn start
```
